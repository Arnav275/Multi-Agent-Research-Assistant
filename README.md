# Multi-Agent-Research-Assistant
Multi-Agent RAG is a QA system using five AI agents to deliver accurate, cited answers from documents. A Planner splits queries, a Router selects document or web search, a Retriever fetches content, a Critic validates results, and a Writer generates the final response. Built with Llama 3.3, Groq, Pinecone, LangGraph, Streamlit, FastAPI, and RAGAS.
